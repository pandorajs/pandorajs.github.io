Pandora 基础设施

  • Class1.1.0

    提供简洁的 OO 实现

  • Events1.1.0

    实现了事件订阅与发布

  • Base1.1.0

    一个基础类，提供 Class、Events 和 Aspect 支持

  • Widget1.1.0

    UI 组件的基础类，约定了组件的基本生命周期，实现了一些通用功能

Pandora 工具

Pandora UI 组件

  • Dialog1.1.0

    Dialog 是通用对话框容器，提供显隐关闭、遮罩层、内容区域自定义以及模态对话框等功能。

  • Select1.1.0

    自定义 select 选择框

  • Switchable1.1.0

    Switchable提供了切换的基本操作, 并通过参数配置实现了自动播放, 循环, 切换等操作, 并提供不同的特点封装成了Tabs, Slide, Carousel模块. 用户可以根据自己的需求引用不同的模块.

  • gallery1.1.0

    组图组件

  • validate1.1.0

    表单验证组件
